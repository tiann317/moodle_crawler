# Moodle Crawler

Download course files in a compressed zip from a Moodle course and external sources.


https://github.com/user-attachments/assets/083c5e6e-e04c-451e-ac8e-e7693183d08b

## What it does

- Downloads files from a Moodle course (PDFs, DOCX, and other files).
- Downloads files from external sources (currently only Google Drive supported).

## Usage

Run the server and open the page in your browser:

```bash
node server.js
```

Then open http://localhost:3000 and follow the prompts to start a download.
